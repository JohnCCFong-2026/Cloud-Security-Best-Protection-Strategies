### Network Security
AWS network security is built on layered protections including Identity and Access Management (IAM), Virtual Private Cloud (VPC) isolation, encryption, monitoring, and DDoS mitigation. The key principle is the shared responsibility model: AWS secures the infrastructure, while you configure and manage security for your applications and data.
